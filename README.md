# 🛒 Retail Analytics End-to-End Project

## 📌 Project Overview
This project focuses on analyzing retail sales data to understand customer behavior and generate actionable business insights. It covers the complete data analytics workflow from data cleaning and modeling to analysis, visualization, and automation.

---

## 🛠️ Tools & Technologies
- *MySQL* – Data cleaning and data modeling  
- *Python (Pandas, NumPy)* – Data analysis and transformation  
- *Power BI* – Dashboard creation and visualization  
- *Git & GitHub* – Version control and project management  

---

## 📅 Project Workflow

### 🔹 Week 1 – Data Engineering (MySQL)
- Imported raw dataset into MySQL  
- Cleaned data (handled missing values, corrected data types)  
- Created revenue column (Quantity × Unit Price)  
- Designed Star Schema (Fact & Dimension tables)  
- Built relationships between tables  

---

### 🔹 Week 2 – Data Analysis (Python)
- Loaded cleaned data using Pandas  
- Performed *RFM Analysis (Recency, Frequency, Monetary)*  
- Segmented customers (Champions, Loyal Customers, At Risk)  
- Conducted *Market Basket Analysis* to identify product combinations  

---

### 🔹 Week 3 – Data Visualization (Power BI)
- Created KPIs (Total Sales, Orders, Customers)  
- Built an interactive dashboard  
- Added time-based (monthly) and region-based (country) analysis  
- Implemented *Row-Level Security (RLS)* for role-based data access  

---

### 🔹 Week 4 – Automation
- Automated the entire workflow using Python  
- Executed multiple scripts using a single automation file  
- Generated final processed datasets for dashboard integration  

---

## 📊 Key Insights
- High-value customers contribute significantly to overall revenue  
- Product combinations identified through Market Basket Analysis help in cross-selling  
- Sales trends vary across regions and time periods
